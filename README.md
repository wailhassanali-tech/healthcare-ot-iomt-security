# Healthcare OT & IoMT Security Research

## Overview

This repository presents my bachelor's research project on cybersecurity challenges within Healthcare Operational Technology (OT) and the Internet of Medical Things (IoMT).

The project explores how increasing connectivity between medical devices, operational technology, and traditional IT systems introduces new cybersecurity risks for healthcare organisations. It combines academic research with a practical proof-of-concept laboratory to examine vulnerabilities, network segmentation, monitoring, and defensive strategies.

---

## Research Objectives

* Analyse cybersecurity challenges affecting Healthcare OT and IoMT environments.
* Evaluate risks associated with interconnected medical systems.
* Examine the impact of weak segmentation and limited monitoring.
* Assess security frameworks and best practices relevant to healthcare.
* Demonstrate selected attack scenarios within a controlled laboratory environment.

## Practical Laboratory

The project includes a proof-of-concept laboratory environment built using virtual machines to simulate healthcare-related systems and demonstrate selected cybersecurity scenarios in a safe and isolated setting.

The laboratory focuses on:

* Internal access risks
* Network segmentation concepts
* Monitoring challenges
* Security observations within simulated environments
